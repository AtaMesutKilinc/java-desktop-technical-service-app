
# Desktop Application With Java Swing - Technical Service App 
- My aim with the technical service project is to complete the control of the new product registration and repaired products in the technical service centers without any confusion.

- On the homepage,there are repaired products and incoming products.A dynamic search is available for 2 tables.You are directed to the relevant page with the customer add , service add and archive buttons.On the customer add page, you can add, update and delete customers. There is also a table with all customers.On the service add page, you can add a service after selecting a customer from the customer table with dynamic search.You can update and delete any service from the service table below.Only the system owner user can login to the system. Its products are examined in four cases. On the archive page, there are products that have been repaired and delivered to the customer.


<i><b>Product Just Arrived status 0</b></br>
<b>Product In Repair status 1</b></br>
<b>Product Has Been Repaired status 2</b></br>
<b>Product Delivered status 3</b></i></br>
